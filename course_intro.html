這堂課你將會學到：

  • 網頁設計，基礎知識
  • HTML 語法，概念實作能力
  • CSS , Javascript 概念與知識
  • 能做出一個基礎的HTML 網頁

什麼是 HTML，為什麼要打造網頁，就一定要先學它？

HTML元素是構建網站的基石。HTML允許嵌入圖像與物件，並且可以用於建立互動式表單，它被用來結構化資訊——例如標題、段落和列表等等，也可用來在一定程度上描述文件的外觀和語意。

當你掌握了HTML，就可以開始規劃並建置你想要的網頁內容 !
